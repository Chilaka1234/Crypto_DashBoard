# UPRO - Ultrapro Blockchain Crypto Dashboard

A modern, responsive crypto blockchain landing page built with **React** and **Vite**, featuring the Ultrapro Blockchain ecosystem, solutions, and community engagement platform.

## Features

- 🚀 **Fast & Modern**: Built with Vite for lightning-fast development and builds
- 💎 **Responsive Design**: Fully responsive layout that works on all devices
- 🎨 **Beautiful UI**: Modern glassmorphic design with smooth animations
- ⚡ **Performance Optimized**: Optimized images with rounded corners and drop shadows
- 🔐 **Blockchain Focused**: Designed specifically for crypto and blockchain projects

## Project Structure

```
src/
├── components/         # React components
│   ├── Navbar/        # Navigation bar with logo
│   ├── Hero/          # Hero section with CTA buttons
│   ├── About/         # About section with images
│   ├── FeatureCard/   # Feature showcase cards
│   ├── EcosystemCard/ # Ecosystem cards with icons
│   ├── SolutionCard/  # Solution cards
│   └── Footer/        # Footer with social links
├── sections/          # Page sections
├── assets/            # Images, icons, and media
├── constants/         # Configuration and constants
├── services/          # API and utility services
└── App.jsx           # Main app component
```

## Tech Stack

- **Frontend**: React 19
- **Build Tool**: Vite 8
- **Styling**: Plain CSS with CSS variables
- **Icons**: React Icons & Lucide React
- **Font**: Poppins (Google Fonts)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Run ESLint
npm run lint
```

## Development

The app is optimized with:

- **Rounded Images**: All images and icons have smooth rounded corners
- **Centered Layout**: Footer and main content are properly centered
- **CSS Variables**: Easy theme customization via CSS custom properties
- **Responsive Grid**: Mobile-first responsive design

## Styling

All styling is handled through CSS files located alongside components:

- Global styles in `index.css` and `App.css`
- Component-specific styles in their respective folders
- Consistent use of CSS variables for colors and spacing

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
